Cara Pemasangan Addon io_scs_tools (Manual)

1. Ekstrak file addon (io_scs_tools.zip) menggunakan aplikasi unzip (WinRAR, 7zip, dsb).
2. Setelah diekstrak, akan muncul folder bernama "io_scs_tools".
3. Buka folder berikut sesuai versi Blender yang digunakan:
   C:\Users\NAMA_USER\AppData\Roaming\Blender Foundation\Blender\2.79\scripts\addons\
   (Ganti "NAMA_USER" dan "2.79" sesuai user dan versi Blender Anda)

4. Drag & drop folder "io_scs_tools" hasil ekstrak ke dalam folder "addons" tersebut.

5. Buka Blender, masuk ke menu Edit > Preferences > Add-ons.
6. Cari "SCS Tools" di kolom pencarian.
7. Centang/aktifkan addon "SCS Tools".

Addon siap digunakan!

Catatan:
- Jika Blender sudah terbuka sebelum pemasangan, restart Blender setelah copy folder addon.
- Untuk versi Blender lain, sesuaikan folder "2.79" dengan versi yang Anda pakai (misal: "2.93", "3.0", dst).












>>>> English <<<<

How to Install io_scs_tools Addon (Manual)

1. Extract the addon file (io_scs_tools.zip) using an unzip application (WinRAR, 7zip, etc).
2. Once extracted, a folder named "io_scs_tools" will appear.
3. Open the following folder according to the Blender version used:
   C:\Users\NAME_USER\AppData\Roaming\Blender Foundation\Blender\2.79\scripts\addons\
 (Replace "NAME_USER" and "2.79" according to your user and Blender version)

4. Drag & drop the extracted "io_scs_tools" folder into the "addons" folder.

5. Open Blender, go to Edit > Preferences > Add-ons.
6. Search for "SCS Tools" in the search field.
7. Check/enable the "SCS Tools" addon.

The addon is ready to use!

Notes:
- If Blender was already open before installation, restart Blender after copying the addon folder.
- For other Blender versions, adjust the "2.79" folder to the version you are using (e.g. "2.93", "3.0", etc.).
